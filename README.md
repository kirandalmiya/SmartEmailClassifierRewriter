## API Key Setup (Important)

1. Create a `.env` file in the root folder.

2. Add your OpenAI key like this:
        OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxx


3. The app uses `python-dotenv` to load the key automatically.
